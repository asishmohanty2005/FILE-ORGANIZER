# Automated File Organizer

Automatically organize messy folders by moving files into category-based subfolders.

## Features
- 📊 **Scan & Analyze** - See what files you have before organizing
- 🚀 **Auto-Organize** - Sort files into Images, Documents, Videos, etc.
- 🔍 **Dry Run Mode** - Preview changes without moving anything
- 🔄 **Undo Support** - Restore files to original locations
- 📁 **Smart Duplicate Handling** - Renames conflicting files
- 🎯 **Custom Categories** - Easy to add new file types

## Project Overview

The File Organizer is a Python-based automation tool designed to organize files into appropriate folders based on their file types. The application scans a selected directory, identifies different file formats, and automatically moves files into categorized folders such as Documents, Images, Videos, Audio, Archives, and Others.

This project helps users maintain a clean and structured workspace while demonstrating practical Python concepts such as file handling, directory management, automation, and exception handling.
