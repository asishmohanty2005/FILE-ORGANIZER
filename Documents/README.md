# Automated File Organizer

Automatically organize messy folders by moving files into category-based subfolders.

## Features
- 📊 **Scan & Analyze** - See what files you have before organizing
- 🚀 **Auto-Organize** - Sort files into Images, Documents, Videos, etc.
- 🔍 **Dry Run Mode** - Preview changes without moving anything
- 🔄 **Undo Support** - Restore files to original locations
- 📁 **Smart Duplicate Handling** - Renames conflicting files
- 🎯 **Custom Categories** - Easy to add new file types

## Installation

No external libraries needed! Just Python 3.6+:

```bash
python main.py
